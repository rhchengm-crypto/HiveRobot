## 左肩前摆 90 度测试

测试目标：前摆到向前约 90 度，保持 5 秒，再回 home。

关键结果：

- home：`0.7120241`
- 90 度位置：约 `-0.8390555`
- 变化量：约 `-1.551 rad`，约 `-88.9°`
- 90 度保持时扭矩：约 `tau = -4.67`
- 回 home 后误差：约 `-0.003815 rad`
- 机身：未明显抖动。

结论：

- 左肩前摆向前 90 度短时保持通过。
- 90 度保持扭矩较高，后续装爪子和拿物体后，需要关注电机温度和持续扭矩。

## 左肩侧摆 90 度测试

## 桌面安全高度与联动姿态测试

测试日期：2026-07-04

测试目的：确认左臂在桌面环境下，后续动作前需要先抬高到的安全前摆角度。

测试动作顺序：

```text
1. 左肩前摆向前 70°
2. 左肩内旋 90°
3. 左肘向前 90°
4. 保持
5. 反顺序回 home
```

方向定义：

```python
front_offset = -math.radians(70)
shoulder_rot_in_offset = math.pi / 2
elbow_offset = math.pi / 2
```

说明：脚本输出文字仍显示“前摆60度”，但从 target 计算，实际前摆为 70°。

关键数据：

```text
home 左肩前摆 = 0.7322423
front target = -0.4894882
差值 = -1.2217305 rad ≈ -70°
```

阶段结果：

| 阶段 | 左肩前摆 pos | target | err | tau | 说明 |
|---|---:|---:|---:|---:|---|
| 前摆 70° | -0.4686427 | -0.4894882 | 0.0208455 | -4.9299 | 左臂刚好抬到桌面之上 |
| 前摆70° + 肩内旋90° | -0.4686427 | -0.4894882 | 0.0208455 | -4.9299 | 肩内旋到位，线束需继续观察 |
| 前摆70° + 肩内旋90° + 肘前90° | -0.4728389 | -0.4894882 | 0.0166493 | -3.9179 | 姿态可保持 |
| 回 home | 0.7276646 | 0.7322423 | -0.0045777 | 0.8684 | 回零可接受 |

其他关节关键结果：

- 左肩内旋 90°到位：`左肩旋转 pos ≈ -0.1459144`，target `≈ -0.1452715`，误差约 `-0.000643 rad`。
- 左肘前 90°到位：`左肘 pos ≈ 0.5788891`，target `≈ 0.5822025`，误差约 `-0.003313 rad`。

结论：

- 左肩前摆 70°是当前桌面环境下的安全高度阈值。
- 在进行肩内旋、肘部弯曲或后续抓取动作前，应先使左肩前摆达到约 70°或更高。
- 该角度不是机械极限，而是基于当前桌面高度、底盘高度和左臂结构的环境约束。

建议软件约束：

```python
LEFT_ARM_TABLE_CLEARANCE_FRONT_MIN = -math.radians(70)
```

后续注意：

- 若桌面高度、机器人底盘高度、肩部安装高度、爪子长度变化，需要重新标定该阈值。
- 前摆 70°保持时扭矩约 `-4.9`，仍属于高负载姿态，不建议无任务长时间保持。

## 抖动现象记录

观察：

- 在未压住底盘时，左肩前摆和左肩侧摆运行时机身有明显抖动。
- 压住底盘后，使用较柔参数测试，机身不再明显抖动。
- 用户观察到：动作有时“越快越不抖，慢反而会抖”。

判断：

- 抖动更像是机身/肩部结构低频共振或底盘约束不足，不是 CAN 通信或电机失控。
- PA6-CF 机身存在一定弹性，肩部高力矩动作会使结构产生微小变形和回弹。
- 慢速运动可能扫到结构固有频率；适当加快反而会越过共振区。

建议：

- 不要单纯继续加大 `kp`。
- 优先加强底盘约束、增加底盘重量、加橡胶脚垫、加强肩部连接刚度。
- 肩前摆/侧摆大角度动作可优先尝试 `3-5s` 完成 90 度，而不是过慢的 `8-12s`。
- 后续做速度扫频测试，找出最不抖的区间。

建议扫频范围：

```python
move_time_list = [12.0, 8.0, 6.0, 4.0, 3.0, 2.5]
```

每次保持相同角度和参数，只改变 `move_time`，记录机身抖动情况。

## 当前结论

- 左臂 5 个 DM4340 装机后小幅动作测试通过。
- 左肩前摆向前 90 度短时保持通过，方向为负方向。
- 左肩侧摆向外 90 度短时保持通过，方向为正方向。
- 当前主要问题不是电机输出不足，而是底盘/机身/肩部结构在特定速度和刚度下会被激振。
- 当前参数可作为“左臂装机身稳定测试参数 v1”，但不是最终训练参数。
