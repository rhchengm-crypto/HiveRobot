# rpo_hardware - 萝博派对 RPO 人形机器人硬件

**中文** | [English](./README.md)

> **项目简介：** 本仓库是 **RPO / Roboto Origin** 人形机器人平台的开源硬件库，包含机械设计文件、PCB 原理图、生产制造资料以及按版本维护的硬件文档。

---

## 📂 仓库总览 (Repository Structure)

本项目按照**硬件版本**进行分层管理，包含 **V1.0** 和 **V2.0** 两套主要设计资料。

### 📊 V1.0 vs V2.0 功能对比

| 特性 | V1.0 (Legacy) | V2.0 (Current / 推荐) |
| :--- | :--- | :--- |
| **机械设计** | 原始设计 | **结构强化版** |
| **硬件架构** | 分立模块 | **高度集成单板** |
| **核心改进** | - | **腰部限位、销钉标定、背板开关** |
| **适用场景** | 老用户维护 | 新用户开发、量产 |

---

### 🔧 V1.0 历史硬件（`atom01_mechanic` / `atom01_pcb`）
**定位：** 原始历史版本，包含早期的机械结构与分立电路板设计。相关目录保留历史命名，用于兼容已有文件和引用。

-   **核心组件：**
    -   **机械部分：** 基础的双足结构设计，包含早期的装配文档和 URDF 文件。
    -   **硬件部分：** 分离式的电源板与通讯板（Roboto_Power, Roboto_Usb2Can）。
-   **适用人群：** 仅限仍在使用第一代硬件的老用户参考。

### 🚀 V2.0 RPO / Roboto Origin 硬件（`roboto_origin_mechanic` / `roboto_origin_pcb`）
**定位：** 当前推荐版本，针对易用性、稳定性和集成度进行了全面重构。

#### ✨ 核心更新亮点
-   **🚀 硬件系统：单板集成化**
    -   **优化：** 将电源管理、通讯接口等模块整合为**一块核心电路板**。
    -   **优势：** 大幅简化了内部走线，减少了连接器故障，提升了系统的稳定性和电磁兼容性。
-   **⚙️ 机械系统：结构强化与标定升级**
    -   **腰部限位：** 新增了**机械限位开关**，确保腰部旋转角度的绝对安全。
    -   **标定方式：** 引入**限位销钉标定**工艺，替代了旧版的通用标定方式
    -   **手臂优化：** 针对旧版手臂件存在的问题进行了修正
    -   **背板开关：** 机器人背板处新增了物理**主控板开关**
-   **适用人群：** 所有新用户、开发者及希望体验最新功能的用户。

## 📚 快速开始 (Quick Start)

请根据你拥有的硬件版本，点击进入对应的目录获取详细资料：

1.  **V2.0 用户（推荐）：**
    -   **机械图纸：** 进入 `V2.0/roboto_origin_mechanic/` 查看装配图与 STL 文件。
    -   **PCB 文件：** 进入 `V2.0/roboto_origin_pcb/` 获取 Gerber 和原理图。
2.  **V1.0 历史用户：**
    -   **机械图纸：** 进入 `V1.0/atom01_mechanic/`。
    -   **PCB 文件：** 进入 `V1.0/atom01_pcb/`。

---

> **注意：** V2.0 与 V1.0 的机械结构和硬件接口**不兼容**，请务必确认版本后再进行组装或采购物料

## 命名说明

本仓库已从历史 Atom01 命名迁移为 `rpo_hardware`。
部分 V1.0 目录和模型文件仍保留历史 `atom01` 名称，作为兼容路径使用。
除非同步更新并验证 CAD、URDF、PCB 和文档引用，否则不建议直接重命名这些路径。
