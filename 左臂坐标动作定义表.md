# 左臂坐标 / 动作定义表

记录日期：2026-07-04

本表基于左臂装机后的实测结果。角度单位采用弧度 `rad`，90 度对应 `math.pi / 2`。

## 实测动作方向

| 动作 | 实测方向 | 代码 offset | 说明 |
|---|---:|---|---|
| 左肩前摆向前 90° | 负方向 | `front_offset = -math.pi / 2` | 前摆向前时，电机位置值减小 |
| 左肩侧摆向外 90° | 正方向 | `side_offset = math.pi / 2` | 侧摆向外时，电机位置值增大 |
| 左肩旋转向外 90° | 负方向 | `shoulder_rot_offset = -math.pi / 2` | 肩旋向外时，电机位置值减小 |
| 左肘向前弯 90° | 正方向 | `elbow_offset = math.pi / 2` | 肘部向前弯曲时，电机位置值增大 |
| 左臂旋转向外 90° | 负方向 | `arm_roll_offset = -math.pi / 2` | 臂旋向外时，电机位置值减小 |

## 姿态相关限位

以下限位不是单关节机械极限，而是在特定姿态下实测出的碰撞边界。规划动作时应优先遵守。

### 桌面安全高度阈值

实测姿态：

```python
front_offset = -math.radians(70)
shoulder_rot_in_offset = math.pi / 2
elbow_offset = math.pi / 2
```

动作顺序：

1. 左肩前摆向前 70°
2. 左肩内旋 90°
3. 左肘向前 90°

测试结论：

- 左肩前摆向前 70°时，整条左臂刚好抬高到桌面之上。
- 后续需要在桌面上方做内旋、肘部动作或抓取动作时，前摆应先到达 70°附近或更高安全高度。
- 在低于该角度时，后续动作可能存在扫到桌面或结构干涉的风险。

关键数据：

| 阶段 | 左肩前摆 target | 左肩前摆 pos | err | tau | 说明 |
|---|---:|---:|---:|---:|---|
| 前摆 70° | -0.489488 | -0.468643 | 0.020845 | -4.93 | 刚好抬到桌面之上 |
| 前摆70° + 肩内旋90° + 肘前90° | -0.489488 | -0.472839 | 0.016649 | -3.92 | 姿态可保持 |
| 回 home | 0.732242 | 0.727665 | -0.004578 | 0.868 | 回零可接受 |

建议软件约束：

```python
LEFT_ARM_TABLE_CLEARANCE_FRONT_MIN = -math.radians(70)
```

说明：

- 该阈值是桌面环境下的实测安全高度，不是电机机械极限。
- 若桌面高度、底盘高度、肩部安装高度或末端爪子结构变化，需要重新测试。
- 当前脚本打印文字仍可能显示“前摆60度”，但本次数据实际对应 70°。

### 左肘前弯 90° + 左肩前摆向前 90°时的侧摆向内限位

姿态：

```python
front_offset = -math.pi / 2
elbow_offset = math.pi / 2
```

在该姿态下，左肩侧摆向内方向为负方向：

```python
side_in_offset = -math.radians(angle_deg)
```

实测结果：

| 侧摆向内角度 | 结果 | 备注 |
|---:|---|---|
| 5° | 安全 | 无碰撞 |
| 10° | 安全 | 无碰撞 |
| 15° | 正好安全 | 接近左肩边界，但未碰撞 |
| 20° | 接近碰撞 | 禁止自动规划到该角度 |

建议软件限位：

```python
LEFT_SHOULDER_SIDE_IN_SAFE_LIMIT = -math.radians(12)
LEFT_SHOULDER_SIDE_IN_HARD_LIMIT = -math.radians(15)
LEFT_SHOULDER_SIDE_IN_FORBIDDEN = -math.radians(20)
```

说明：

- `-12°` 用作自动动作、训练、规划的安全限位。
- `-15°` 是人工复测确认的硬上限。
- `-20°` 已接近碰撞，应禁止自动进入。
- 该限位只适用于 `左肘≈+90°` 且 `左肩前摆≈-90°` 附近的姿态。

## 使用注意

- 当前方向定义只适用于当前机械安装方向和电机 ID 配置。
- 左肩前摆、左肩侧摆是当前最容易激发机身抖动的两个关节。
- 左肩侧摆 90°和左肩前摆 90°都属于高负载姿态，短时测试可行，不建议长时间静态保持。
- 后续安装爪子、负载或重新装配结构后，需要重新确认 90°方向和保持扭矩。
